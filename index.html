RI 2025 to 2025 Migration

Source appliance

Run these steps in technician mode.
Stop the RapidIdentity service on the source appliance before creating the export archive.
  1. Download and extract the migration bundle.
cd /var/tmp
rm -rf ri2025-migration-bundle ri2025-migration-bundle.tar.gz
curl -fL -o ri2025-migration-bundle.tar.gz "https://ri-docs-files.s3.us-east-1.amazonaws.com/ri2025-migration-bundle.tar.gz"
tar -xzf ri2025-migration-bundle.tar.gz
cd ri2025-migration-bundle
chmod 700 *.sh
  1. Create the export archive in /var/sftp/files.
./export-appliance-2025.sh --out-file=ri-export.tar.gz
  1. Confirm the archive exists.
ls -lh /var/sftp/files/ri-export.tar.gz

The target appliance will pull files/ri-export.tar.gz from the source by sftp during migration.

Target appliance preparation

Run these steps in technician mode.

Before running migration, update the hosts file on each target appliance so idautodb resolves correctly.

DB appliance

Add idautodb to the localhost entry. Example:

127.0.0.1 localhost idautodb

Non-DB appliance

Add the DB appliance IP entry. Example:

<db-ip> idautodb

Target DB appliance

  1. Download and extract the migration bundle.
cd /var/tmp
rm -rf ri2025-migration-bundle ri2025-migration-bundle.tar.gz
curl -fL -o ri2025-migration-bundle.tar.gz "https://ri-docs-files.s3.us-east-1.amazonaws.com/ri2025-migration-bundle.tar.gz"
tar -xzf ri2025-migration-bundle.tar.gz
cd ri2025-migration-bundle
chmod 700 *.sh
  1. Run migration and include the database migration flag.
./migrate-appliance-2025.sh --src-host=SOURCE_HOST --src-sftp-file=files/ri-export.tar.gz --migrate-db

When prompted for the sftp password, enter the password.

Target non-DB appliance

  1. Download and extract the migration bundle.
cd /var/tmp
rm -rf ri2025-migration-bundle ri2025-migration-bundle.tar.gz
curl -fL -o ri2025-migration-bundle.tar.gz "https://ri-docs-files.s3.us-east-1.amazonaws.com/ri2025-migration-bundle.tar.gz"
tar -xzf ri2025-migration-bundle.tar.gz
cd ri2025-migration-bundle
chmod 700 *.sh
  1. Run migration without the database migration flag.
./migrate-appliance-2025.sh --src-host=SOURCE_HOST --src-sftp-file=files/ri-export.tar.gz

When prompted for the sftp password, enter the password.
